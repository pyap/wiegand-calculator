# wiegand-calculator (26 bit)
```
./hid <dec>
./hid <fac,num>
./hid <fac> <num>
./hid 0x<hex>
./hid 0b<bin>
./hid p:<proxmark tag-id>
./hid b:<proxmark bin>

Ex:
./hid 12070178
./hid 184,11554
./hid 184 11554
./hid 0x3705a44
./hid 0b11011100000101101001000100
./hid p:200775a44
./hid b:111011100000101101001000100
```
